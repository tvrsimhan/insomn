# insomn
### Text editor made with C++ and ncurses
It works on Linux terminal and is easy to use.
 
**Commands:**

|  Commands |Actions   |
| :------------: | :------------: |
| <kbd>X</kbd> + <kbd>Enter</kbd>  |Quit   |
|  <kbd>S</kbd> + <kbd>Enter</kbd>   | Save  |
| <kbd>I</kbd> + <kbd>Enter</kbd>    |  Insert |
| <kbd> Esc</kbd>| Normal|
